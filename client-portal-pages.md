# Client Portal Pages

## Goal

Create the first React screens for Grupo Studio's client portal:

- Login page
- Authenticated home/dashboard page

## Stack

- React
- Axios
- Lucide React
- React Router
- React Query

## UI Direction

Follow the attached layout style only:

- Dark left navigation
- Light work area
- Gold accents
- Compact operational dashboard
- Rounded metric cards

## Scope

- Static front-end prototype with local mock data
- Responsive layout for desktop and mobile
- API client and React Query provider prepared for real endpoints
